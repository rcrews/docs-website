Check query execution

You can determine if query vectorization occurred during execution by checking the log.

The Hive client records information, including whether or not query vectorization occurs, in the log at the info level by default.
  1. Navigate to the location of the Hive log.
    • Navigate to default log location /tmp/<user.name>.
    If the logs are not located in the default location, find the location by looking up the value of hive.querylog.location in hive-site.xml.
  2. Open the log, and look for the query vectorization record.
